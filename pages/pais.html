Decline-in-Value Request for Review

Online Form


The California Revenue and Taxation Code allows for a temporary reduction in assessed value when property suffers a "decline-in-value." A decline-in-value occurs when the market value of your property is less than the assessed value as of January 1, 2015. If a decline-in-value has occurred, the property owner can request a review. This application MUST be filed between July 2 and November 30, 2015.
